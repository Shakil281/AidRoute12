STEP 1: Get your Personal Access Token
--------------------------------------
1. Go to: https://github.com/settings/tokens
2. Click: "Generate new token (classic)"
3. Name: aidroute-deploy
4. Select: repo (check all repo permissions)
5. Generate and COPY the token

==================================================================

STEP 2: Run these commands in PowerShell
----------------------------------------

cd c:\aidroute-frontend

# Push to GitHub (replace YOUR_TOKEN with the token you copied)
git push https://YOUR_TOKEN@github.com/Shakil281/AidRoute12.git main

# OR use this if you want to enter credentials interactively:
git push -u origin main

When prompted:
- Username: Shakil281
- Password: (paste your token)

==================================================================

STEP 3: Verify
--------------
Check: https://github.com/Shakil281/AidRoute12

You should now see all your files!

==================================================================
